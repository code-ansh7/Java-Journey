## What I Learned

* File class is used to work with files in Java.
* Creating a File object does not create an actual file.
* createNewFile() creates a real file in the system.
* exists() checks whether a file exists or not.
* getName() returns the file name.
* getAbsolutePath() returns the complete file location.
* createNewFile() may throw IOException.
* throws IOException can be used to handle checked exceptions.
* File handling is used for permanent data storage.

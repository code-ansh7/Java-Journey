## 📌 Key Learnings

- FileReader reads one character at a time.
- read() returns an integer (ASCII value).
- EOF is represented by -1.
- File pointer moves automatically after every read().
- Once EOF is reached, the FileReader cannot read again.
- A new FileReader object is required to read the file from the beginning.
- Character class methods work on single characters.
- String methods work on complete strings.

# ✍️ File Writing in Java

This folder contains programs related to writing data into files using Java's `FileWriter` class.

## 📚 Programs

### 1️⃣ FileWritingDemo.java
Writes simple text into a file.

### 2️⃣ MultipleLineWriter.java
Writes multiple lines into a file using `\n`.

### 3️⃣ CharacterWriter.java
Writes individual characters into a file.

### 4️⃣ ASCIIWriter.java
Demonstrates writing ASCII values using:

```java
write(int)
```

Example:

```java
write(65);
```

Output:

```text
A
```

### 5️⃣ InputDataWriter.java
Takes file content from the user and stores it inside a file.

### 6️⃣ FileWritingRevision.java
Complete revision program combining:

- File Creation
- File Information
- User Input
- File Writing
- Success Messages

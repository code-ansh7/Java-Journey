### What I Learned

* ArrayList is a dynamic array in Java.
* ArrayList automatically grows when new elements are added.
* add() is used to insert elements.
* get() is used to access elements using indexes.
* set() is used to update existing elements.
* remove() is used to delete elements.
* size() returns the total number of elements.
* contains() checks whether an element exists in the list.
* isEmpty() checks whether the list is empty.
* clear() removes all elements from the list.
* ArrayList can store both strings and numbers.
* nextLine() is required after nextInt() to clear the input buffer.
* The last valid index is always size() - 1.
* ArrayList is more flexible than traditional arrays.

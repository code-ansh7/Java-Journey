### What I Learned

* HashSet stores only unique elements.
* Duplicate elements are automatically ignored.
* HashSet does not guarantee insertion order.
* add() is used to insert elements.
* contains() is used to search elements.
* remove() is used to delete elements.
* size() returns the total number of unique elements.
* isEmpty() checks whether the HashSet is empty.
* clear() removes all elements from the HashSet.
* HashSet can be traversed using a for-each loop.
* HashSet is useful when duplicate data is not allowed.
* HashSet is commonly used for usernames, emails, IDs and mobile numbers.
* User input can be stored directly in a HashSet.
* HashSet provides fast searching and duplicate checking.

### What I Learned

* HashMap stores data in Key-Value pairs.
* Keys must be unique.
* Duplicate keys replace old values.
* Duplicate values are allowed.
* put() is used to insert key-value pairs.
* get() is used to access values using keys.
* containsKey() checks whether a key exists.
* containsValue() checks whether a value exists.
* remove() removes a key-value pair.
* size() returns total key-value pairs.
* isEmpty() checks whether the HashMap is empty.
* clear() removes all key-value pairs.
* keySet() returns all keys.
* values() returns all values.
* HashMap can be traversed using keySet() and get().
* HashMap is useful for storing structured data such as names, marks, usernames and passwords.

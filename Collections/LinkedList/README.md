### What I Learned

* LinkedList stores elements using nodes.
* Each node contains data and a reference to the next node.
* LinkedList is different from ArrayList because elements are not stored in continuous memory locations.
* add() is used to insert elements.
* get() is used to access elements using indexes.
* remove() is used to delete elements.
* size() returns the total number of elements.
* addFirst() inserts an element at the beginning.
* addLast() inserts an element at the end.
* getFirst() returns the first element.
* getLast() returns the last element.
* LinkedList supports traversal using loops.
* LinkedList is useful when frequent insertion and deletion operations are required.
* LinkedList can be used to model real-world structures such as queues and train coaches.
